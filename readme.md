# Flask Project

## What is used:
* python = "3.7.3"
* Flask = "1.1.2"
* waitress = "1.4.4"
* poetry = " 1.1.4"
## How run:
1. Get git clone in console: "git clone https://github.com/IvanDumych/PP_Project.git"
2. Using pyenv create interpretor
```
pyen install 3.7.3
pyenv local 3.7.3
```
3. Run Poetry:
```
pip install poetry
poetry run python wsgi.py
```
